## Termux-os

* [+] Based on Zsh Shell !

* [+] Pure Termux Look !



## Installation

* `termux-setup-storage`

* `apt update && apt upgrade`

* `pkg install git -y`

* `git clone https://github.com/aviyt31622/Avi-os.git`

* `cd Avi-os`

* `chmod +x *`

* `bash os.sh`
